Markdown Basics

Text Formatting

Italics
*This text will be italic*
This text will be italic

Bold
**This text will be bold**
This text will be bold

Strikethrough
~~This text will be crossed out~~
This text will be crossed out

Links
[This site](https://www.porybox.com) will make "This site" clickable.

Quotes
Use > at the beginning of the line to display the text in a block quote.
> I have been on many Pokémon sites but Porybox is the best one.

I have been on many Pokémon sites but Porybox is the best one.

Images

Displaying images
You can display images hosted on other sites directly in your notes. Use the following syntax:
![description of the image](link to the image)
![Porybox logo](https://www.porybox.com/img/logo-header.png)
Images will be displayed with a maximum width of 600 pixels.

Lists

Unordered lists
For a bulleted list of items, use * at the beginning of the entries.
* Item 1
* Item 2
* Item 3

Ordered lists
For a numbered list of items, use n. at the beginning of the entries, where n is an integer.
1. Item 1
2. Item 2
3. Item 3

Headers

Headers
Use # at the beginning of a line for a top-level header. Markdown supports 6 levels of subheaders.
# Level 1
### Sublevel 3
###### Sublevel 6

Tables

Tables
This code:
First Header | Second Header
:------------ | :-------------
Content row 1 column 1 | Content row 1 column 2
Content row 2 column 1 | Content row 2 column 2

Yields a table like the following:

First Header Second Header
Content row 1 column 1 Content row 1 column 2
Content row 2 column 1 Content row 2 column 2
